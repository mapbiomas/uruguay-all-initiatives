# UY Uruguay — Official GitHub Repositories 🌎

This section aggregates **MapBiomas Uruguay** repositories dedicated to thematic collections, cross-cutting themes, mosaics, validation workflows, and analytical modules.

All repositories listed below are maintained by the **MapBiomas Uruguay** team and follow the official **MapBiomas methodological framework**.

---

## 🇺🇾 MapBiomas Uruguay

Official repositories responsible for the production, processing, validation, and analysis of MapBiomas Uruguay data.

### 🗂️ Thematic Collections

Repositories dedicated to the generation of thematic collections for land use and land cover mapping in Uruguay.

### 🔎 Cross-cutting Themes

Repositories focused on transversal themes and integrated analytical components across MapBiomas Uruguay products.

### 🛰️ Mosaics

Repositories containing scripts for mosaic generation, preprocessing, and satellite data organization.

### 📊 Validation Workflows

Repositories designed for validation, accuracy assessment, and methodological consistency checks.

### 🧩 Analytical Modules

Repositories with scripts and tools for analysis, interpretation, and data exploration.
